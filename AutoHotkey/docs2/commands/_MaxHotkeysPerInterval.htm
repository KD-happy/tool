#最大热键频率

与 #热键频率 一起使用, 指定热键激活的速率, 超过此速率会显示警告对话框.

#最大热键频率 Value

参数

Value

类型: 整数

在 #热键频率 指定的间隔内, 不触发警告对话框情况下可以按下热键的最大数目.

备注

注意不要把上面的数值设置的过于宽松, 因为如果您可能曾经不小心引入了无限循环的键击(通过会意外触发其他热键的 发送 函数), 那么您的电脑会因快速的键击事件流而无响应.

举个简单的例子, 热键 ^c::发送 "^c" 会产生无限循环的键击. 要避免此问题, 添加 $ 前缀到热键的定义(例如 $^c::), 这样 发送 函数就不会触发热键了.

如果脚本中未使用此指令, 那么它会表现的就像此设置为 70 那样.
