变量 := expression

计算表达式并把结果保存到变量.

变量 := expression

参数

Var

保存 表达式 结果的变量的名称.

Expression

有关详情, 请参阅表达式和下面的示例.

注意, 此参数不是可选的. 若要清空该变量, 请指定一个空字符串. 例如: x := ""

备注

单词 true 和 false 是值分别为 1 和 0 的内置变量. 可以使用它们使脚本更易于阅读, 如以下示例所示:

CaseSensitive := false
ContinueSearch := true

可以使用这个函数和任何其他接受 OutputVar 的函数来创建伪数组. 这可以通过让 OutputVar 包含另一个变量的引用来实现, 例如 数组%i% := Var/100 + 5. 有关更多信息, 请参阅数组.
