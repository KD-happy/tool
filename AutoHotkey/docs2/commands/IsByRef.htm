是否传址

如果指定的 ByRef 参数作为变量提供, 则返回一个非零数字.

是否传址 := 是否传址(ParameterVar)

参数

ParameterVar

类型: 变量

变量的引用. 例如: 是否传址(MyParameter).

返回值

类型: 整数(布尔值)

如果 ParameterVar 是 ByRef 参数, 且调用者传递了这个参数, 则函数返回 1(true); 否则当 ParameterVar 为其他类型参数时返回 0(false).

相关

ByRef 参数

示例

#1

信息框 Function(MyVar)

Function(ByRef Param)
{
    返回 是否传址(Param)
}
